# Run Instructions for Pavicon Website

This is a static HTML/CSS website. You can run it simply by opening the HTML files in your browser.

## Prerequisites

- A modern web browser (Chrome, Firefox, Edge, etc.)

## How to Run

1.  Navigate to the project directory: `c:\Users\Admin\Documents\GitHub\PaviconWebsite`
2.  Double-click on `index.html` to open the homepage.
3.  Navigate through the links to view other pages.

## Editing

To edit the website, you can use any text editor or IDE (like VS Code).
- **CSS:** The main styles are in `assets/css/style.css` and `assets/css/pavicon.css`.
- **HTML:** Each page has its own HTML file (e.g., `index.html`, `about.html`, etc.).
- **Images:** Images are located in the `assets/img/` directory.

## Build

Since this is a static site, no build step is required. However, if you want to deploy it, you can simply upload the `PaviconWebsite` folder to your web server.
